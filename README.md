## Installation

The package can be installed with the Julia package manager. From the Julia REPL, type `]` to enter the Pkg REPL mode and run:

````julia
pkg> add AustralianLandscapeWaterBalance
````

If you want to install the package directly from its github development site,

````julia
pkg> add http://github.com/petershintech/AustralianLandscapeWaterBalance.jl
````

And load the package using the command:

````julia
using AustralianLandscapeWaterBalance
````
